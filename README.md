📅 Gestión de Citas Técnicas (PHP & SQLite)
Desarrollado por: Miguel Netti
Perfil: Técnico Superior Universitario en Informática

Este proyecto nace de la necesidad de gestionar servicios técnicos (mantenimiento de hardware y software) de manera eficiente. Permite registrar clientes, asignar servicios y, lo más importante, pactar un precio personalizado en el momento de la cita.

🛠️ Habilidades Técnicas Demostradas
Backend: Desarrollo de lógica de servidor con PHP 8.x.

Base de Datos: Diseño relacional y manipulación de datos con SQLite 3.

Frontend Interactivo: Uso de JavaScript para actualizar precios en tiempo real sin recargar la página.

UI/UX: Interfaz limpia con Bootstrap 5 e iconografía profesional de Bootstrap Icons.

Seguridad: Implementación de sentencias preparadas (PDO) para prevenir ataques de Inyección SQL.

🚀 Funcionalidades
Registro de Citas: Formulario completo con validación de campos obligatorios (nombre, servicio, fecha y hora).

Control de Precios: Los servicios tienen un precio base, pero el sistema permite editar el "Precio Pactado" manualmente.

Tablero de Control: Visualización de citas con estados (Pendiente/Completado) y acciones de edición/eliminado.

Resumen Financiero: Cálculo automático del "Total Proyectado" basado en las citas agendadas.

🔧 Instalación Local
Clona el repositorio: git clone https://github.com/tu-usuario/gestion-citas.git

Asegúrate de tener PHP configurado en tu variable de entorno (PATH).

Inicia el servidor: php -S localhost:8000

Accede a: http://localhost:8000
